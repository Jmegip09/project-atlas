# Project Atlas

Project Atlas is an enterprise-style supply chain intelligence platform that evolves my CS-499 Inventory Management System capstone into a full data engineering and analytics project.

The goal is to simulate how a modern organization could track inventory, purchase orders, suppliers, warehouse activity, and operational KPIs using SQL, Python, PostgreSQL, Power BI, and eventually cloud/data warehouse tools.

## Project Origin

Project Atlas began as my CS-499 Computer Science Capstone: an Android/Kotlin inventory management system refactored using MVVM, Room, DAO layers, repository pattern, secure coding practices, and optimized search/filter logic.

Atlas expands that foundation into a broader platform focused on supply chain analytics, data quality, and business intelligence.

## Core Business Questions

- Which products are at risk of stockout?
- Which suppliers are underperforming?
- Which warehouses have receiving delays?
- Where are inventory discrepancies occurring?
- Which purchase orders are delayed?
- What is the total inventory value by warehouse and product category?
