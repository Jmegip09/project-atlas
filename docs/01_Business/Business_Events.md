# Business Events

## Event 1

Purchase Order Created

Generates:

- Purchase Order Number
- Buyer
- Supplier
- Order Date

---

## Event 2

Shipment Received

Generates:

- Receiving Record
- Warehouse
- Employee
- Quantity Received

---

## Event 3

Inventory Adjustment

Generates:

- Adjustment Record
- Reason Code
- Quantity Difference
- Timestamp

---

## Event 4

Cycle Count

Generates:

- Count Session
- Counted Quantity
- Variance
- Employee
